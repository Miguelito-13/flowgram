HOW TO MAKE A FLOWCHART

How to drag and drop

Only the START/END, INPUT/OUTPUT, PROCESS and DECISION symbols can be dragged and dropped on the canvas.

Note: The pointer icon must be active in order to drag and drop a symbol.

  1. Click on the pointer icon to make sure it is active.
  2. Click and hold on to the symbol from the left.
  3. Move the symbol to the desired spot on the canvas.
  4. Release the symbol.

How to connect a symbol to another symbol.

Note: The pointer icon must be active to enable the connector.

  1. Click on the connector symbol.
  2. Click on the source symbol or the symbol where the path will start.
  3. Click on the blank space adjacent to the source symbol.
  4. Click on the path where it will point to the destination symbol.
  5. If there are any mistakes in drawing the path of the arrow, click again on the highlighted path to remove it.
  6. If the destination symbol is clicked, it will automatically generate the symbol based on the highlighted path.

For Decision symbol as the starting symbol:

  1. Click on the connector symbol.
  2. Click on the decision symbol.
  3. A popup box will appear and choose if the path is TRUE or FALSE.
  4. Draw the path leading to the destination symbol.
  5. Click again on the connector symbol and then on the decision symbol if there are multiple results on the decision.

How to edit symbol text

Only the symbols that are in the canvas can be edited. Text on the connectors cannot be edited.

  1. Click on the Text icon.
  2. Click on the symbol on which the text will be changed.
  3. Type the text that will be contained in the symbol
  4. Press Enter when finished editing.

How to delete a symbol or arrow/connector

Only the symbols and connectors on the canvas can be deleted.

  1. Click on the trash icon.
  2. Click on the desired symbol or connector that will be deleted.
  3. If the symbol has a connector, the connector will also be removed.
  4. If the user clicked on a part of the connector, the whole connector will be removed

How to run the flowchart

  1. If the flowchart is complete, click on the Run button
  2. If there are any errors on the flowchart, the symbol that contains the error will be highlighted and the output window will prompt an error message.
  3. If there are user inputs required, the input window is where it will accept inputs.
  4. If the execution is stuck on a loop, the stop button will terminate the execution.
  5. The prowchart will finish compiling and running if it doesn't encounter any error.

PROPER SYNTAX/STRUCTURE

Flowchart Symbols

Symbol Description Syntax
Start and End of the flowchart START or END
Designates the direction of the flowchart
Assigns values to variables and displays the value. INPUT x
OUTPUT y
OUTPUT "Hello ", z, a
Contains a step in the process x = 0
z = y + 9
c = a % c
The outcome of this symbol affects the direction of the flowchart x <= 10
y >= 20 OR (z > 0 AND a == 10)
  • The INPUT/OUTPUT symbol must not contain any mathematical or logical expressions.
  • To get the INPUT from the user, the variable must be declared first inside the process symbol
  • For OUTPUT, the string must be declared first before concatinating a variable.
  • For mathematical expressions, the prowchart follows the PMDAS rule.
  • The DECISION symbol must not contain any mathematical expressions.
  • The connectors cannot overlap with each other but it can be treated also like a destination symbol. It will follow the direction on where the arrow is headed.
